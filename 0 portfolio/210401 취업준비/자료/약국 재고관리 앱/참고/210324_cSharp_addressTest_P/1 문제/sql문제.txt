※ 아래의 수행 조건을 참조하여 SQL 활용 평가를 수행하고 결과를 화면 캡처하시오.

[수행 조건] 
1. 테이블 이름은 namecard 로사용하고 각 칼럼 정보는 아래와 같다.
- id 는 숫자이며 값이 무조건 필요하고 자동으로 값이 증가한다.
- name 은 문자열이며 크기는 10, 값이 무조건 존재한다. 
- tel 은 문자열이며 크기는 20, 값이 무조건 존재하다. 
- email 은 문자열이며 크기는 30, 값이 없을 수도 있다. 
- address 는 문자열이며 크기는 50, 값이 무조건 존재한다. 
- id 는 반드시 테이블에서 유일한 값을 가져야만 한다. 

2. 테이블에 3개의 서로 다른 값들을 삽입하는 SQL문을 작성하시오. 
- 단, name 속성의 값은 ‘홍길동', ‘김길동', ‘박길동'으로 사용하시오.

3. 이름의 끝자리가 ‘동'으로 끝나는 모든 이름을 검색하고 id 가 내림차순으로 정렬하시오. 
4. 김길동의 이름을 검색하여 전우치로 변경하고 name이 오름차순으로 정렬하시오. 
5. 전우치 이름을 검색하여 정보를 삭제하시오. 
6. namecard를 복사하여 namecard_copy 테이블을 생성하시오.
- 단, 김길동 데이터만 namecard_copy에 복사되게 하시오.
